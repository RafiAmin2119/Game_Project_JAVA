# 🎮 Java TicTacToe Game

A simple 2-player Tic-Tac-Toe game made with Java Swing. The project was created to practice basic Object-Oriented Programming (OOP) concepts and Java GUI design in a fun and interactive way.

The game includes a 3×3 board, score counting, multiple rounds, and winner messages. OOP concepts like Encapsulation, Inheritance, Abstraction, and Polymorphism were used to keep the code clean and organized.
